IPipe

Notes · Fact-Check · Translate

Primary AI (Required)

Watches YouTube videos directly. Required for all features.

Optional — Cross-Fact-Check AIs

Add one or both to cross-validate facts from multiple AI perspectives.

Uses GPT-4o with web search for real-time fact-checking.

Uses Claude Opus for deep reasoning-based fact analysis.

Optional — Indian Language Translation

Translates notes & summaries to Hindi, Tamil, Telugu, Kannada, Bengali, and more.

Quick Start

  1. Get a Gemini key from Google AI Studio
  2. Optionally add OpenAI / Claude keys for cross-fact-checking
  3. Optionally add a Sarvam AI key for Indian language translation
  4. Go to any YouTube video — the AI panel appears on the right
